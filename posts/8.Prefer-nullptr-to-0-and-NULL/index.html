0кіј NULLліҙлӢӨ nullptrлҘј м„ нҳён•ҳлқј

Posted Feb 7, 2024
By Yang jeong hyeon
6 min read

  • мҡ”м җл¶Җн„° л§җн•ҳл©ҙ лҰ¬н„°лҹҙ 0мқҖ intмқҙкі  нҸ¬мқён„°к°Җ м•„лӢҲлӢӨ.

  • нҸ¬мқён„°л§Ң мӮ¬мҡ©н•  мҲҳ мһҲлҠ” мң„м№ҳм—җ 0мқҙ мһҲмңјл©ҙ C++мқҖ null нҸ¬мқён„°лЎң н•ҙм„қн•ҳм§Җл§Ң, мқҙлҠ” мөңнӣ„мқҳ мҲҳлӢЁмқҙлӢӨ

  • C++мқҳ кё°ліём Ғмқё л°©м№ЁмқҖ, 0мқҖ intмқҙм§Җ нҸ¬мқён„°к°Җ м•„лӢҲлқјлҠ” кІғмқҙлӢӨ.


NULL

  • мӢӨмҡ©м Ғмқё кҙҖм җм—җм„ңлҠ” NULLлҸ„ л§Ҳм°¬к°Җм§Җм§Җл§Ң, лӢӨмҶҢ л¶Ҳнҷ•мӢӨн•ң м„ёл¶Җ мӮ¬н•ӯмқҙ мЎҙмһ¬н•ңлӢӨ.

  • нқ”м№ҳ м•Ҡм§Җл§Ң, м»ҙнҢҢмқјлҹ¬к°Җ NULLм—җ int мқҙмҷёмқҳ м •мҲҳ нҳ•мӢқмқ„ л¶Җм—¬н•  мҲҳлҸ„ мһҲкё° л•Ңл¬ёмқҙлӢӨ

  • C++98м—җм„ң мқҙ м җмқҙ м•јкё°н•ҳлҠ” мЈјлҗң л¬ём ңлҠ” нҸ¬мқён„°мҷҖ м •мҲҳ нҳ•мӢқм—җ лҢҖн•ң мҳӨлІ„лЎңл”©мқҙ мқҳмҷёмқҳ л°©мӢқмңјлЎң н•ҙмҶҢлҗңлӢӨ.

            // f(void*)лҘј нҳём¶ңн•ҳлҠ” кІҪмҡ°лҠ” м—ҶлӢӨ

  • NULLмқҖ 0L(longмқҳ 0)мңјлЎң м •мқҳлҗҳм–ҙ мһҲлӢӨл©ҙ мқҙ нҳём¶ңмқҖ мӨ‘мқҳм ҒмқҙлӢӨ

    • long->int, long->bool, long->void* лӘЁл‘җ мҡ°м„ мҲңмң„к°Җ к°ҷмқҢ

  • мқҙ нҳём¶ңмқҳ л¬ём ңм җмқҖ мҶҢмҠӨмҪ”л“ңмқҳ мҷёкҙҖмғҒ мқҳлҜёмҷҖ мӢӨм ң мқҳлҜёк°Җ лӢӨлҘҙлӢӨлҠ” кІғмқҙлӢӨ.

    • л„җ нҸ¬мқён„°лЎң fлҘј нҳём¶ңн•ҳлҠ”кІҢ м•„лӢҢ м •мҲҳлЎң fлҘј нҳём¶ңн•ҳлҠ”кІҢ лҗҳм–ҙлІ„лҰј


nullptr

  • nullptrмқҖ NULLкіј лӢӨлҘҙкІҢ м •мҲҳ нҳ•мӢқлҸ„ м•„лӢҲкі  к·ёл ҮлӢӨкі  нҸ¬мқён„° нҳ•мӢқлҸ„ м•„лӢҲлӢӨ.

  • nullptrлҠ” лӘЁл“  нҳ•мӢқмқҳ нҸ¬мқён„°лқјкі  мғқк°Ғн•ҳл©ҙ лҗңлӢӨ.

    • nullptrмқҳ мӢӨм ң нҳ•мӢқмқҖ std::nullptr_tмқёлҚ°, мқҙ нғҖмһ…мқҖ nullptr нҳ•мӢқмңјлЎң м •мқҳлҗңлӢӨ.

    • nullptr_tлҠ” лӘЁл“  raw нҸ¬мқён„° нҳ•мӢқмңјлЎң м•”л¬өм Ғ ліҖнҷҳмқҙ к°ҖлҠҘн•ҳл©°, мқҙ л•Ңл¬ём—җ nullptrмқҖ лӘЁл“  нҳ•мӢқмқҳ нҸ¬мқён„°мІҳлҹј н–үлҸҷн•ңлӢӨ.

  • м•һм—җ лӮҳмҳЁ f н•ЁмҲҳлҘј nullptrлЎң м „лӢ¬н•ңлӢӨл©ҙ м •мҲҳ л§Өк°ңліҖмҲҳк°Җ м•„лӢҢ нҸ¬мқён„° нҳ•мӢқмқҳ л§Өк°ңліҖмҲҳк°Җ нҳём¶ңлҗңлӢӨ.

н…ңн”ҢлҰҝм—җм„ңмқҳ nullptr

  • nullptrмқҖ н…ңн”ҢлҰҝм—җм„ң лҚ”мҡұ нҷңм•Ҫн•ңлӢӨ.

  • м Ғм Ҳн•ң л®Өн…ҚмҠӨлҘј мһ к·ј мғҒнғңм—җм„¬лӮҳ нҳём¶ңн•ҙм•ј н•ҳлҠ” н•ЁмҲҳк°Җ мһҲлҠ”лҚ°, к·ё н•ЁмҲҳл“Өмқҙ лӢӨмқҢмІҳлҹј к°Ғмһҗ лӢӨлҘё мў…лҘҳмқҳ нҸ¬мқён„°лҘј л°ӣлҠ”лӢӨкі  к°Җм •н•ңг„·лӢӨ.

template<typename FuncType, typename MuxType, typename PtrType>
decltype(auto) lockAndCall(FuncType func, MuxType& mutex, PtrType ptr)
{
    using MuxGuard = lock_guard<MuxType>;

    MuxGuard g(mutex);
    return func(ptr);
}

  • мІ« нҳём¶ңмқё 0мқ„ л„ҳкІЁмЈјл©ҙ м»ҙнҢҢмқјлҹ¬к°Җ 0мқҳ нҳ•мӢқмқ„ нҢҢм•…н•ҳкё° мң„н•ҙ н…ңн”ҢлҰҝ нҳ•мӢқ м—°м—ӯмқ„ м Ғмҡ©н•ҳлҠ”лҚ°, 0мқҳ нҳ•мӢқмқҖ н•ӯмғҒ intлқјлҠ” кІғмқҙлӢӨ.

    • л§Өк°ңліҖмҲҳ ptr нҳ•мӢқмқҖ intк°Җ лҗҳкі , int нҳ•мӢқмқҖ shared_ptr<Widget>кіј нҳёнҷҳлҗҳм§Җ м•Ҡм•„ м»ҙнҢҢмқј мҳӨлҘҳ

  • л‘җ лІҲм§ё нҳём¶ңмқё NULL м—ӯмӢң 0кіј к°ҷмқҙ int нҳ•мӢқмңјлЎң м—°м—ӯлҗҳм–ҙ, unique_ptr<Widget>кіј нҳёнҷҳлҗҳм§Җ м•Ҡм•„ м»ҙнҢҢмқј мҳӨлҘҳк°Җ л°ңмғқн•ңлӢӨ.

  • л°ҳл©ҙ, nullptrл§Ң нҳём¶ңм—җ л¬ём ңк°Җ м—Ҷкі  ptrмқ„ f3м—җ м „лӢ¬н•ҳл©ҙ std::nullptr_tм—җм„ң Widget*лЎңмқҳ м•”л¬өм Ғ ліҖнҷҳмқҙ мқјм–ҙлӮңлӢӨ.


кІ°лЎ 

  • 0кіј NULLмқҖ м •мҲҳ нҳ•мӢқмңјлЎңлҸ„ лҗҳкё° л•Ңл¬ём—җ м• л§ӨлӘЁнҳён•ң мғҒнҷ©мқҙ л°ңмғқн•ҳкұ°лӮҳ мқҳлҸ„м№ҳ м•ҠмқҖ кІ°кіјк°Җ мқјм–ҙлӮ  мҲҳ мһҲм§Җл§Ң nullptrлҠ” н•ӯмғҒ нҸ¬мқён„° нҳ•мӢқмқ„ к°ҖлҰ¬нӮӨкё° л•Ңл¬ём—җ мқҙлҹ¬н•ң л¬ём ңл“Өмқ„ н•ҙкІ°н•  мҲҳ мһҲлӢӨ.

    • 0кіј NULLмқҖ м •мҲҳ,нҸ¬мқён„° л‘ҳ лӢӨ к°ҖлҠҘн•ҳм—¬ мҳӨлІ„лЎңл”© л¬ём ңм—җ лҜјк°җн•ҳм§Җл§Ң, nullptrмқҖ к·ёл Үм§Җ м•ҠлӢӨ.

мқҙкІғл§ҢмқҖ мһҠм§Җ л§җмһҗ!
