н•ӯлӘ©3 : лӮҢмғҲл§Ң ліҙмқҙл©ҙ constлҘј л“ӨмқҙлҢҖ ліҙмһҗ!

  • const нӮӨмӣҢл“ңлҠ” мҶҢмҠӨ мҪ”л“ңм—җм„ң вҖҳмқҳлҜём Ғмқё м ңм•ҪвҖҷ(const нӮӨмӣҢл“ңк°Җ л¶ҷмқҖ к°қмІҙлҠ” мҷёл¶Җ ліҖкІҪмқ„ л¶Ҳк°ҖлҠҘн•ҳкІҢ н•ңлӢӨ.)мқ„ кұёкІҢ н•ҙмЈјкі  м»ҙнҢҢмқјлҹ¬к°Җ мқҙ м ңм•Ҫмқ„ м§ҖмјңмӨҖлӢӨ.

  • const нӮӨмӣҢл“ңлҠ” нҒҙлһҳмҠӨ л°”к№Ҙм—җм„ңлҠ” м „м—ӯ нҳ№мқҖ л„Өмһ„мҠӨнҺҳмқҙмҠӨ мң нҡЁлІ”мң„мқҳ мғҒмҲҳлҘј м„ м–ё(м •мқҳ)н•ҳлҠ” лҚ° мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

  • к·ё лҝҗл§Ң м•„лӢҲлқј нҢҢмқј, н•ЁмҲҳ, мң нҡЁлІ”мң„м—җм„ң staticмңјлЎң м„ м–ён•ң к°қмІҙм—җлҸ„ constлҘј л¶ҷмқј мҲҳ мһҲлӢӨ.

  • нҒҙлһҳмҠӨ лӮҙл¶Җмқҳ кІҪмҡ°м—җлҠ”, м •м Ғ л©ӨлІ„ л°Ҹ л№„м •м Ғ лҚ°мқҙн„° л©ӨлІ„ лӘЁл‘җлҘј мғҒмҲҳлЎң м„ м–ён•  мҲҳ мһҲлӢӨ.

  • нҸ¬мқён„°лҠ” нҸ¬мқён„°к°Җ к°ҖлҰ¬нӮӨлҠ” лҚ°мқҙн„° нҳ№мқҖ нҸ¬мқён„° к·ё мһҗмІҙлҘј мғҒмҲҳлЎң м§Җм •н•  мҲҳ мһҲлӢӨ.

  • м–ҙл–Ө л°ҳліөмһҗлҘј constлЎң м„ м–ён•ҳлҠ” мқјмқҖ нҸ¬мқён„°лҘј constлЎң м„ м–ён•ҳлҠ” кІғ(мҰү, T* const нҸ¬мқён„°)кіј к°ҷлӢӨ.

  • л°ҳліөмһҗлҠ” мһҗмӢ мқҙ к°ҖлҰ¬нӮӨлҠ” лҢҖмғҒмқҙ м•„лӢҢ кІғмқ„ к°ҖлҰ¬нӮӨлҠ” кІҪмҡ°лҠ” м•Ҳлҗҳм§Җл§Ң, л°ҳліөмһҗк°Җ к°ҖлҰ¬нӮӨлҠ” лҢҖмғҒ мһҗмІҙлҠ” ліҖкІҪмқҙ к°ҖлҠҘн•ҳлӢӨ.

  • л§Ңм•Ҫ ліҖкІҪмқҙ л¶Ҳк°ҖлҠҘн•ң к°қмІҙлҘј к°ҖлҰ¬нӮӨлҠ” л°ҳліөмһҗ(мҰү, const T* нҸ¬мқён„°мқҳ STL лҢҖмқ‘л¬ј)к°Җ н•„мҡ”н•ҳлӢӨл©ҙ const_iteratorлҘј мӮ¬мҡ©н•ҳл©ҙ лҗңлӢӨ.

  • constмқҳ к°ҖмһҘ к°•л Ҙн•ң мҡ©лҸ„лҠ” н•ЁмҲҳ м„ м–ём—җ мӮ¬мҡ©н•  л•ҢмқҙлӢӨ.

  • н•ЁмҲҳ м„ м–ёл¬ём—җ мһҲм–ҙ constлҠ” н•ЁмҲҳ л°ҳнҷҳ к°’, к°Ғк°Ғмқҳ л§Өк°ңліҖмҲҳ, л©ӨлІ„ н•ЁмҲҳ м•һм—җ л¶ҷмқ„ мҲҳ мһҲкі , н•ЁмҲҳ м „мІҙм—җ лҢҖн•ҙ const м„ұм§Ҳмқ„ л¶ҷмқј мҲҳлҸ„ мһҲлӢӨ.

  • н•ЁмҲҳ л°ҳнҷҳ к°’мқ„ constлЎң л§Ңл“Өл©ҙ, м•Ҳм •м„ұмқҙлӮҳ нҡЁмңЁмқ„ нҸ¬кё°н•ҳм§Җ м•Ҡкі лҸ„ мӮ¬мҡ©мһҗмёЎмқҳ м—җлҹ¬ мғҒнҷ©мқ„ мӨ„мқҙлҠ” нҡЁкіјлҘј мһҗмЈј ліј мҲҳ мһҲлӢӨ.

мғҒмҲҳ л©ӨлІ„ н•ЁмҲҳ

  • л©ӨлІ„ н•ЁмҲҳм—җ л¶ҷлҠ” const нӮӨмӣҢл“ңлҠ” н•ҙлӢ№ л©ӨлІ„ н•ЁмҲҳк°Җ мғҒмҲҳ к°қмІҙм—җ лҢҖн•ҙ нҳём¶ңлҗ  н•ЁмҲҳмқҙлӢӨлқјлҠ” мӮ¬мӢӨмқ„ м•Ңл ӨмЈјлҠ” кІғмқҙлӢӨ.

  • мқҙ кІғмқҙ мӨ‘мҡ”н•ң мқҙмң лҠ” л‘җк°Җм§Җк°Җ мһҲлӢӨ.

  • мІ« м§ё, нҒҙлһҳмҠӨмқҳ мқён„°нҺҳмқҙмҠӨлҘј мқҙн•ҙн•ҳкё° мўӢкІҢ н•ҳкё° мң„н•ЁмқҙлӢӨ.,

    • к·ё нҒҙлһҳмҠӨлЎң л§Ңл“Өм–ҙм§„ к°қмІҙлҘј ліҖкІҪн•  мҲҳ мһҲлҠ” н•ЁмҲҳк°Җ л¬ҙм—Үмқём§Җ,лҳҗ ліҖкІҪн•  мҲҳ м—ҶлҠ” н•ЁмҲҳлҠ” л¬ҙм—Үмқёк°ҖлҘј мӮ¬мҡ©мһҗмёЎм—җм„ң м•Ң мҲҳ мһҲлӢӨ.

  • л‘ҳ м§ё, const нӮӨмӣҢл“ңлҘј нҶөн•ҙ мғҒмҲҳ к°қмІҙлҘј мӮ¬мҡ©н•  мҲҳ мһҲкІҢ н•ңлӢӨ.

    • C++ н”„лЎңк·ёлһЁмқҳ мӢӨн–ү м„ұлҠҘмқ„ лҶ’мқҙлҠ” н•өмӢ¬ кё°лІ• мӨ‘ н•ҳлӮҳмқёмғҒмҲҳ к°қмІҙм—җ лҢҖн•ң м°ёмЎ°мһҗ(reference-to-const)лЎң м§„н–үн•ҳлҠ” кІғмқҙкё° л•Ңл¬ёмқҙлӢӨ.

    • мң„мқҳ кё°лІ•мқ„ мӮ¬мҡ©н•ҳкё° мң„н•ҙм„  мғҒмҲҳ мғҒнғңлЎң м „лӢ¬лҗң к°қмІҙлҘј мЎ°мһ‘н•  мҲҳ мһҲлҠ” const л©ӨлІ„ н•ЁмҲҳ, мҰү мғҒмҲҳ л©ӨлІ„ н•ЁмҲҳк°Җ мӨҖл№„лҗҳм–ҙ мһҲм–ҙм•ј н•ңлӢӨлқјлҠ” кІғмқҙлӢӨ.


  • const нӮӨмӣҢл“ңк°Җ мһҲкі  м—Ҷкі мқҳ м°Ёмқҙл§Ң мһҲлҠ” л©ӨлІ„ н•ЁмҲҳл“ӨмқҖ мҳӨлІ„лЎңл”©мқҙ к°ҖлҠҘн•ҳлӢӨ.

class TextBlock{
public:
    // const к°қмІҙм—җ лҢҖн•ң operator[]
    const char& operator[] (std::size_t position) const 
    { return text[position]; }

    // non-const к°қмІҙм—җ лҢҖн•ң operator[]
    char& operator[] (std::size_t position)
    { return text[position]; }

private:
    std::string text;
};

// мӮ¬мҡ© мҳҲмӢң

// TextBlock::operator[]мқҳ non-const н•ЁмҲҳ нҳём¶ң
TextBlock tb("Hello");
std::cout << tb[0];     

// TextBlock::operator[]мқҳ const н•ЁмҲҳ нҳём¶ң
const TextBlock ctb("World");
std::cout << ctb[0];


  • operator[]лҘј мҳӨлІ„лЎңл“ңн•ҙм„ң к°Ғ лІ„м „л§ҲлӢӨ л°ҳнҷҳ нғҖмһ…мқ„ лӢӨлҘҙкІҢ н–Ҳкё° л•Ңл¬ём—җ,TextBlockмқҳ мғҒмҲҳ к°қмІҙмҷҖ л№„ мғҒмҲҳ к°қмІҙмқҳ м“°мһ„мғҲк°Җ лӢ¬лқјм§„лӢӨ.

    • мғҒмҲҳ лІ„м „мқҳ operator[]мқҳ н•ЁмҲҳк°Җ л°ҳнҷҳ нғҖмһ…мқҙ constм—¬м„ң л°ңмғқн•ң м—җлҹ¬

  • лҳҗ н•ҳлӮҳ лҲҲм—¬кІЁ ліј м җмқҖ operator[]мқҳ non-const л©ӨлІ„ н•ЁмҲҳлҠ” charмқҳ м°ёмЎ°мһҗлҘј л°ҳнҷҳн•ҳлҠ” кІғмқёлҚ°, м°ёмЎ°к°Җ м•„лӢҲлқјл©ҙ 2лІҲм§ё мӨ„мқҙ м—җлҹ¬к°Җ лӮңлӢӨ.


  • л№„нҠё мҲҳмӨҖ мғҒмҲҳм„ұ(bitwise constness)мқҖ м–ҙл–Ө л©ӨлІ„ н•ЁмҲҳк°Җ к·ё к°қмІҙмқҳ м–ҙл–Ө лҚ°мқҙн„° л©ӨлІ„лҸ„ кұҙл“ңлҰ¬м§Җ м•ҠкІ лӢӨлҠ” мҰү,к·ё л©ӨлІ„ н•ЁмҲҳк°Җ constмһ„мқ„ мқём •н•ҳлҠ” к°ңл…җ

  • л…јлҰ¬м Ғ мғҒмҲҳм„ұ(logical constness)лҠ” л№„нҠё мҲҳмӨҖ мғҒмҲҳм„ұм—җм„ң лІ—м–ҙлӮң кІҪмҡ°лҘј ліҙмҷ„н•ҳлҠ” к°ңл…җ

    • мҳҲлҘј л“Өм–ҙ, мғҒмҲҳ к°қмІҙмқҳ нҸ¬мқён„°лҘј лӢӨлҘё нҸ¬мқён„°м—җ л°ӣм•„м„ң ліҖкІҪн•ҳлҠ” кІҪмҡ°


  • const н•ЁмҲҳ л©ӨлІ„ м•Ҳм—җм„ңлҠ” л©ӨлІ„ ліҖмҲҳлҘј ліҖкІҪн•  мҲҳ м—Ҷм§Җл§Ң mutable нӮӨмӣҢл“ңлҘј мӮ¬мҡ©н•ҳл©ҙ к°ҖлҠҘн•ҙм§„лӢӨ.
