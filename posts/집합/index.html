집합

Posted Jan 12, 2024
By Yang jeong hyeon
3 min read

집합

  • 집합에 관련된 것들을 쉽게 처리하기 위해 만들어진 자료형

  • 집합 자료형은 set 키워드를 통해 만들 수 있다.

1
2
3
4

s1 = set([1,2,3])

s1
# {1,2,3}
  • 위와 같이 set()의 괄호 안에 리스트를 입력하여 만들거나 문자열을 입력할 수도 있다.
1
2
3
4

s2 = set("Hello")

s2
# {'e','l','o','H'}


집합 자료형의 특징

  • 위의 set 문자열의 결과를 보면 l이 두개였지만 하나만 들어가고 순서도 제각각인데 이것은 set의 특징때문이다.

  • set의 특징

    • 중복을 허용하지 않는다.

    • 순서가 없다(unordered)

  • 리스트와 튜플은 순서가 있어 인덱싱을 통해 자료형 값을 얻을 수 있지만
    set 자료형은 순서가 없어 인덱싱을 사용하지 못한다.

  • set의 자료형에 저장된 값을 인덱싱으로 접근하려면 다음과 같이 리스트나 튜플로 변환한 후 해야한다.

리스트

1
2
3
4
5
6
7
8

s1 = set([1,2,3])
l1 = list(s1) # set을 리스트로 변환

l1
# [1,2,3]

l1[0]
# 1

튜플

1
2
3
4
5
6
7

t1 = tuple(s1) # set을 튜플로 변환

t1
# (1,2,3)

t1[0]
# 1


집합 자료형 활용

  • set 자료형이 유용하게 사용되는 경우는 교집합,합집합,차집합을 구하는 경우이다.

  • 먼저 다음과 같이 set 자료형 2개가 있다고 가정한다.

1
2

s1 = set([1,2,3,4,5,6])
s2 = set([4,5,6,7,8,9])

교집합

  • 두 set 변수의 교집합은 & 기호를 이용하여 구할 수 있다.
1
2

s1 & s2
# {4,5,6}
  • intersection 함수를 사용해도 동일한 결과를 리턴한다.
1
2

s2.intersection(s1)
# {4,5,6}

합집합

  • 합집합은 | 기호를 이용하여 구할 수 있다.
1
2

s1 | s2
# {1,2,3,4,5,6,7,8,9}
  • union 함수로 구할 수도 있다.
1
2

s1.union(s2)
# {1,2,3,4,5,6,7,8,9}

차집합

  • 차집합은 - 기호를 이용해서 구할 수 있다.
1
2
3
4
5

s1 - s2
# {1,2,3}

s2 - s1
# {8,9,7}
  • difference를 이용해서 구할 수도 있다.
1
2
3
4

s1.difference(s2)
# {1,2,3}
s2.difference(s1)
# {8,9,7}


집합 관련 함수

add

  • set 자료형에 값을 추가할 수 있다.
1
2
3
4
5

s1 = set([1,2,3])
s1.add(4)

s1
# {1,2,3,4}

update

  • set 자료형에 한꺼번에 여러 값을 추가할 수 있다.
1
2
3
4
5

s1 = set([1,2,3])
s1.update([4,5,6])

s1
# {1,2,3,4,5,6}

remove

  • set 자료형에서 특정 값을 지울 수 있다.
1
2
3
4
5

s1 = set([1,2,3])
s1.remove(2)

s1
# {1,3}
