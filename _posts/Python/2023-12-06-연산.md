# **연산**

```py
# 더하기
1 + 2

# 빼기
5 - 4

# 곱하기
7 * 6

# 나누기
9 / 3

# 제곱
2 ** 3 
```

* 연산 처리 순서

```py
# ()

```

# **반올림,올림,내림**

## **반올림**

* 반올림 함수는 `round` 를 사용

```py
print(round(8/3))
print(round(7/3))

# 출력
# 3
# 2
```

* round 함수의 `두 번째 매개변수에 반올림할 자릿수를 결정`할 수 있다.


```py
print(round(8/3, 2))

# 출력
# 2.67
```

## **내림**

* 내림은 `//` 키워드 혹은 `floor` 를 사용

```py
print(floor(8/3))
print(8 // 3)
```

## **올림**

* 올림은 `ceil`를 사용

```py
print(ceil(8/3))
```
