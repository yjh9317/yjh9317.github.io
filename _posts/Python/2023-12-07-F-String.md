# **F-String**

* F-String은 문자열 앞에 `f`를 붙이는 것으로, 다른 변수와 연동할 수 있다.


```py
score = 0
height = 1.8
isWinning = True

# f-String
print(f"your score is {score}")
print(f"your hegiht is {height}")
print(f"you are winning is {isWinning}")

'''
your score is 0
your height is 1.8
you are winning is True
'''
```
