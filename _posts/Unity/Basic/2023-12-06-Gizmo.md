# **Gizmo**

* Gizmo는 디버깅을 목적으로 특정 도형을 그리는 것

* `OnDrawGizmos`함수에서 `Gizmos` 클래스를 통해 원하는 도형을 만들 수 있다.

* 아래는 선을 그리는 기즈모

```c#
private void OnDrawGizmos()
{
    // 시작 위치, 끝 위치
    Gizmos.Drawline(StartPosition, EndPosition);
}
```
